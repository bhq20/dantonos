Backup / Sincronização

Para usar os mesmos dados em vários dispositivos: exportar de um, importar no outro.

Instalar no iPhone/iPad

1. Abra esse arquivo no Safari (não no app do OneDrive direto)
2. Toque em ⬆ Compartilhar na barra inferior
3. Role e toque em "Adicionar à Tela de Início"
4. Confirme — o DANTON OS aparece como app na sua tela

Importante: só assim os dados ficam salvos permanentemente no iPhone/iPad. Abrir direto pelo OneDrive perde os dados ao fechar.

Como abrir do OneDrive no Safari

Se o arquivo está no OneDrive:
1. Abra o app OneDrive · toque no arquivo
2. Toque ⋯ (três pontinhos) → "Abrir em..." → Safari
3. Depois "Adicionar à Tela de Início" (passos acima)

Alternativa: salve este arquivo no app Arquivos do iPhone e abra dali.
